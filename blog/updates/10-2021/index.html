Обновления за октябрь 2021г

  • Neiros
  • Блог
  • Обновления

Вячеслав Власенко

01.12.2021

Работа в новой карточке клиента

  • Дополнительные поля контактов. Добавлена возможность добавлять неограниченное количество полей контактной информации.
  • Сумма сделки по частям. Разбивайте добавление платежей в сделках на любое количество частей.
  • История визитов. Просматривайте информацию по истории визитов наших клиентов.

Интеграции

  • amoCRM. Добавили возможность пробрасывать в дополнительное поле карточки заявки источник обращения.
  • Facebook. Перешли на новую версию API платформы.
  • МойСклад. Новая логика названия сделок при создании заказа в платформе.

Телефония

  • Поочередная переадресация. Настраивайте поочередную переадресацию на несколько своих номеров в сценариях коллтрекинга и виджета коллбек. С возможностью ставить время в секундах, через которое переводить на следующий номер.
  • Объединение записей в журнале. При звонках с переадресацией на несколько номеров произвели объединение записей в журнале звонков.
  • Точность коллтрекинга.  Доработали определение времени резерва номера за событиям движения мыши. Теперь время резерва обновляется при любом движении пользователя.

А так же

  • Почтовый клиент. Улучшен алгоритм определения номера телефона клиента из текста входящих e-mail писем.
  • События.Вставка скрытого поля hiddenl с параметром neiros_visit в формы на сайте.
  • Уведомления. Реализованы уведомления о текущих сделках и коммуникаций за сутки.
